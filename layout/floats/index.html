Floats for layout

Floats are often used for page layout

Say we want to make a traditional layout with our main content taking up 2/3rds of our page and a right hand side-bar taking up the remaining 1/3rd

We might use the following code inside our <body> tag

    
<div class="layout">
    <article>
        <h1>Our pages main content</h1>
        <p>This is where we are putting our pages main content</p>
    </article>
    <aside>
        <h2>Secondary content</h2>
        <p>This is where we are putting supplementary content.</p>
    </aside>
</div>

So to layout our content as described above we float both <article> and <aside> left and give them the approriate widths.

    
article,
aside {
    float: left;
}
article {
    width: 66.66667%;
}
aside {
    width: 33.33333%;
}

So to layout our content as described above we float both <article> and <aside> left and give them the approriate widths.

I'm using percentage widths to make a fluid layout which is the first step to responsive design.

Clearfix

We can see that our pink background has dissapeared.

This is because of the float parent collapse issue described in the floats page

So we need to apply our clear fix to <div class="layout">

If we are using the HTML5 boilerplate we can do this with the clearfix class like so <div class="layout clearfix">

Full HTML

    
<div class="layout clearfix">
    <article>
        <h1>Our pages main content</h1>
        <p>This is where we are putting our pages main content</p>
    </article>
    <aside>
        <h2>Secondary content</h2>
        <p>This is where we are putting supplementary content.</p>
    </aside>
</div>

Gutters

Now we just need to add some padding or margin to our <article> and <aside> to separate them.

Remember if you are using Border-box sizing than padding and border will stay within your declared width, but margin is outside width.

So you can add padding without changing width and thus you can safely use a pixel value for the padding if you like.

If you use margins to add spacing, you'll need to subtract the appropriate amount of width from one or both of your <article> or <aside>

I'm going to add a margin-right to article of 2.08333% (which is 20px when layout container is 960px wide). I'm going to subtract this from my <article> width.

    
article,
aside {
    float: left;
}
article {
    width: 64.58334%; /* 66.66667% - 2.08333% */
    margin-right: 2.08333%;
}
aside {
    width: 33.33333%;
}
