Paragraphs are everwhere <p></p>

Typically you might want to set the following CSS styles for headings and paragraphs:

font-family: (you would set the default family on the body tag though)
font-size: (best to use em units - remember 1em equals 1 x basefont size which by default is 16px)
line-height: (this is how you can increase your leading, but its not leading its the height of the line! The paragraph sits vertically in the middle of the line.)
margin: (usually just top and bottom)
color:

These examples to the left are just default styling (after normailize.css applied)

P.S. I don't recommend setting font-size: on parent containers eg <div>, <body> unless you really want to step up or step down all font sizes for that particular area. Because em's multiply, so if I set font-size: 2em for <body> and font-size: 1.2em for <p> then my paragraphs will be 16p * 2 * 1.2 = 38.4px.

I'm sure you can see where this might be useful, eg to reduce base font-size for <footer>, but also potentiall cause frustration.
