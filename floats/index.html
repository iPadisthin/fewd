The HTML code for this is:

NOTE: I've also wrapped this in a div with pink background so you can see what's happening.

The CSS to float the image left is:


img {
    float: left; 
    margin-right: 20px;
}

The margin right is required to give some spacing on the right of the image.

See how the image is still in its same vertical position! It's just floating now so the following <p> tag now moves up to occupy the space, but the <p>'s content wraps around the image.

See how the second image is to the left of the first even though it comes second in the HTML.

This is because like the text, it is wrapping to the left of the first image, and because it is also floating right, the text in turn wraps around it.

What if we want the second image to sit under the first image, but we still want the text to wrap around both images.

Well then we need to add clear: right; to the CSS for the second image so that it clears the float of the first image!

Let's give each image a class so we can assign seperate CSS.

The HTML:

    
<p>Lorem ipsum...</p>
<img class="image1" src="http://placehold.it/350x150">
<img class="image2" src="http://placehold.it/150x150">
<p>Morbi nisi...</p>
<p>Duis gravida...</p>

The CSS


img {
    float: right; 
    margin-left: 20px;
}
img.image2 {
    clear: right;
}

    float: right;

}
img.image2 {
    clear: right;
}

The parent collapse

So you can hopefully see something happening in the the last two examples.

The parent div with pink background is not visually wrapping all its children.

This is because the floated images do not contribute to the height of the parent div, only the paragraphs are.

This would be no problem if the parent's div background was the same as the pages background and we also wanted any content after the div to also wrap around the image for example:

    
<div class="div1">    
    <p>Lorem ipsum...</p>
    <img class="image1" src="http://placehold.it/350x150">
    <img class="image2" src="http://placehold.it/150x150">
    <p>Morbi nisi...</p>
    <p>Duis gravida...</p>
</div>
<div class="div2">    
    <p>Lorem ipsum...</p>
    <p>Morbi nisi...</p>
</div>
