Buttons (and how you can use multiple classess)

So buttons are fun.

There is a <button> tag that looks like this and you can style it.

But a lot of buttons on websites are really just links, they look like a button but they are really just a call to action, rather than a form button or a button that makes something happen via javascript.

So lets's make a call to action button

We'll use an <a> tag with class="button"

The HTML

<a class="button">Call to action</a>

Now for the CSS:

.button {
    display: inline-block;
    padding: 15px 20px;
    margin: 5px;
    font-size: 1.5em;
    background: #178CA6;
    color: white;
    text-decoration: none; /* gets rid of underline on link */
    text-shadow: 0 2px 0 #0F5B6C;
    border-radius: 6px;
    box-shadow: 0 3px 0 #0F5B6C;
    cursor:pointer; /* we need this because we lose the pointer when we make an a tag display block */
}
.button:hover {
    background: #147990;
}

Here it is: Call to action do you like it? How would you style your button?

Now lets say we want a red button for a different action.

We can use multiple classes instead of defining everything again.

<a class="special-offer button">Call to action</a>

.special-offer.button {
    background: #F24130;
    text-shadow: 0 2px 0 #99160A;
    box-shadow: 0 3px 0 #99160A;
    
    text-transform: uppercase;
}
.special-offer.button:hover {
    background: #C01B0C;
}
