How to host a website on GitHub

For your Final Project you will use GitHub to host your website!

Here are the steps for setting this up!

  1. Step 1: Login to GitHub website and create a new repository

    Click on green "+ New repository" button on your GitHUb homepage

    Insert name of repository and click on green "Create repository" button.

    Name your repository whatever you think is appropriate for your site, what you call it will be in the URL which will look like this: http://yourgithubusername.github.io/repositoryname

    Remember no spaces in repo name (best to use lower case and no characters that need shift key).

  2. Step 2: Setup your repository on your desktop (your computer)

    Click on grey "Setup in Desktop" button

    Your GitHub for desktop application should launch and look something like below. Choose the folder you wish to add your repo clone to and click "Clone" button.

  3. Step 3: Creat a branch named "gh-pages"

    In your GitHub desktop app click on the "Add a branch" button (highlighted in image below) and enter "gh-pages" for the name and click on blue "Create branch" button. It must be gh-pages

  4. Step 4: Add a file to your repository

    Create an index.html file in the repository folder on your computer (create it in your text editor).

  5. Step 5: Commit and Publish index.html file

    In your GitHub desktop type a summary, click commit and then publish.

  6. Step 6: Check your website

    Open a new tab in your browser and enter URL http://yourgithubusername.github.io/repositoryname

    In the case of my example http://ipadisthin.github.io/website/

  7. Step 7: Build your site

    Now you just need to build your site, commit and sync to add changes.

Note: using / in absolute paths on git hub pages sites will take you back to http://yourgithubusername.github.io/ so you'll always need to follow / with the repository name i.e. /repositoryname/ , e.g. for a link <a href="/website/path/to/file.html">
